Proyecto N°1:

Se desarrollo un software para controlar las finanzas de una empresa. Permitiendo entender cómo se distribuyen y utilizan los recursos la misma, para así optimizar el rendimiento de estos capitales

Proyecto N°2:

Aplicación de ECommerce para un local de productos electrónicos. Esta aplicación cuenta con catalogo, carrito, pagos integrados para tarjetas internacionales y la opción de realizar envios.

Proyecto N°3:

Página Web a medida para un portal de noticias internacional. Este sitio cuenta con la posibilidad de suscripciones a la página, para leer las noticias por un monto determinado.

Proyecto N°4:

Se implemento un software para la tesorería de una importante empresa. Este software monitorea los gastos y genera gráficos/estadísticas según los datos obtenidos. También sugiere mejoras para optimizar los recursos.

Proyecto N°5:

Se implemento un sitio web a medida para un arquitecto reconocido a nivel internacional. Se dio especial énfasis en crear una galería en la cual se pudieran visualizar las obras en las cual estuvo implicado el arquitecto.

Proyecto N°6:

Se diseño un software capaz de controlar varios dispositivos simultáneamente desde la PC. Este programa brinda la posibilidad de interconectar todos los dispositivos vinculados a la misma red.

Proyecto N°7:

Web para plataforma educativa orientada a dictar cursos relacionados con la programación. La plataforma cuenta con una sección teórica (en vivo o pre grabadas) y otra sección practica para a poner a prueba los conocimientos.

Proyecto N°8:

Plataforma educativa dedicada a impartir una amplia variedad de cursos en diferentes idiomas. Cuenta con sistema de suscripción y pagos internacionales para formarse desde cualquier parte del mundo.

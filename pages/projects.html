Proyecto N°1:

Se desarrollo un software para controlar las finanzas de una empresa. Permitiendo entender como se distribuyen y utilizan los recursos de una empresa, para así optimizar el rendimiento de estos capitales

Proyecto N°2:

Aplicacón de ECommerce para un local de productos electronicos. Esta aplicación cuenta con catalogo, carrito, pagos integrados para tarjetas internacionales y la opción de realizar envios

Proyecto N°3:

Pagina Web a medida para un portal de noticias internacional. Esta web cuenta con la posibilidad de suscripciones a la pagina, para leer las noticias por un monto determinado.

Proyecto N°4:

Se implemento un software para la tesorería de una importante empresa. Este software monitorea los gastos y genera graficos/estadisticas según los datos obtenidos. También sugiere mejoras para optimizar los recursos

Proyecto N°5:

Se implemento una web a medida para un arquitecto reconocido a nivel internacional. Se dio especial enfasis en crear unia galeria en la cual se pudieran visualizar las obras en las cual estuvo implicado el arquitecto

Proyecto N°6:

Se diseño un software capaz de controlar varios dispositivos simultaneamente desde la PC. Este programa brinda la posibilidad de interconectar todos los dispositivos vinculados a la misma red.

Proyecto N°7:

Web para plataforma educativa orientada a dictar cursos online relacionados con la programación. Esta plataforma cuenta con una sección teorica (en vivo y pre grabadas) y otra sección orientada a poner a prueba los conocimientos.

Proyecto N°8:

Plataforma educativa dedicada a impartir una amplia variedad de cursos en diferentes idiomas. Cuenta con sistema de suscripción y pagos internacionales para formarse desde cualquier parte del mundo.
